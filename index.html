    So, what's up with Minima? (a big post, 14/07/2012)

    Minima 1.0.1 was released 20 days ago, and it fixed some bugs and probably added some more. It also added an update system, but that’s IRRELEPHANT.

    When will 1.1 or 2.0 be developed?

    I don’t know yet, actually. I don’t even know what will be in the version, but I (with the help of awesome people from the linuxzasve.com forum), chose the codenames for 2.0 and a few version after, and they are, in no particular order: Dracula, Dalek, and Darby. (If you wondered, 0.6 was Cheesy, 1.0 didn’t have a codename, and 1.0.1 was Chihuahua)

    So, what’s up with the logo?

    The logo changed a lot in this three versions, and the 1.0/1.0.1 version was this: https://github.com/iwebhub/minima-cms/raw/master/img/logo.png

    But, the trouble starts with the demo logo: http://minimademo.net84.net/img/logo.png

    It uses a different font (original was Georgia), and it’s quite different from the version above.

    There will be a new logo in 2.0, and if you have a suggestion, leave a photo reply to this post. I will design one too, so we’ll have a battle. The text should be Minima or Minima CMS.

    About fonts: Use only free/opensource/public domain fonts, and don’t use sites like dafont. A better place to start is Google Webfonts.

    Your logo must be public domain, and don’t use any trademarks. (TM, R, C symbols are forbidden)
